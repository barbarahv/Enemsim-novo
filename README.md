# ENEMSIM NOVO

Hybrid system for ENEMSIM:
- **Web**: Next.js (React)
- **Mobile**: React Native (Expo)
- **Backend**: Node.js + Express
- **Database**: PostgreSQL

## Structure
- `backend/`: API server
- `web/`: Frontend PWA
- `mobile/`: iOS/Android App

## Getting Started
1. **Backend**: `cd backend && npm install && npm start`
2. **Web**: `cd web && npm run dev`
3. **Mobile**: `cd mobile && npx expo start`
